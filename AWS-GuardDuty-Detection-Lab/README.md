# AWS GuardDuty Detection Lab

## Project Overview
This project demonstrates how AWS GuardDuty can detect suspicious activities and improve threat visibility in an AWS environment.

## Services Used
- AWS GuardDuty
- IAM
- CloudTrail
- EC2

## Key Findings
- Enabled GuardDuty
- Reviewed security findings
- Investigated alerts
- Improved cloud visibility

## Lessons Learned
Visibility is critical in cloud security. GuardDuty helps identify threats that traditional preventive controls may miss.
