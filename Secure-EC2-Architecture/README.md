## Project Overview
Built and secured an EC2 architecture in AWS using security best practices.

## What I Implemented
- Created a secure EC2 instance
- Configured Security Groups
- Restricted SSH access
- Used key pair authentication
- Tested inbound traffic rules
- Connected to the instance using EC2 Instance Connect

## Security Concepts Practiced
- Principle of least privilege
- Network access control
- Secure remote access
- AWS Security Groups

## Challenges Faced
Initially, my EC2 instance would not connect when restricted to my IP address. After troubleshooting networking and inbound rule configurations, I successfully connected securely.

## Tools Used
- AWS EC2
- Security Groups
- VPC
- VS Code
- GitHub

## Key Takeaway
Security is not just deploying resources — it is configuring access properly while maintaining usability.
