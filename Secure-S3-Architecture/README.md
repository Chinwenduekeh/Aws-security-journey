# Secure S3 Architecture on AWS

## Overview
Designed and implemented a secure Amazon S3 architecture following AWS security best practices.

## AWS Services Used
- Amazon S3
- IAM
- AWS CloudTrail
- AWS KMS
- CloudWatch

## Security Features
- Block Public Access
- Server-side encryption
- Least privilege IAM access
- Bucket policies
- Logging and monitoring


## Lessons Learned
- Importance of least privilege
- How S3 bucket policies work
- AWS monitoring and logging fundamentals
